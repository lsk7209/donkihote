# 애드센스 승인 체크리스트

## ✅ 완료된 항목

### 1. 필수 페이지 (Required Pages)

- ✅ **About 페이지** (`/about`)
  - 서비스 개요, 주요 기능, 서비스 목적, 특징 등 상세한 내용 포함
  - 연락처 정보 및 관련 링크 제공
  - SEO 최적화된 메타데이터

- ✅ **Contact 페이지** (`/contact`)
  - 명확한 연락처 정보 (이메일)
  - 문의 유형별 안내
  - 응답 시간 안내
  - 개인정보 보호 안내

- ✅ **Privacy Policy** (`/privacy`)
  - 개인정보 처리 목적
  - 개인정보 처리 및 보유기간
  - 처리하는 개인정보 항목
  - **쿠키 정책** (필수)
  - **Google AdSense 관련 내용** (필수)
  - 개인정보 제3자 제공
  - 정보주체의 권리
  - 개인정보 파기
  - 개인정보 보호책임자

- ✅ **Terms of Service** (`/terms`)
  - 약관의 적용
  - 용어의 정의
  - 서비스 제공 및 변경
  - 이용자의 의무
  - 지적재산권
  - 서비스 이용 제한
  - 면책조항
  - 분쟁의 해결
  - 약관의 변경
  - 문의처

### 2. 네비게이션 (Navigation)

- ✅ **Header 네비게이션**
  - 도구, FAQ, 소개, 문의 링크 포함
  - 모든 필수 페이지 접근 가능

- ✅ **Footer 네비게이션**
  - 법적 정보 섹션 (개인정보 처리방침, 이용약관, 소개, 문의)
  - 서비스 섹션 (FAQ, 사용법 가이드)
  - 저작권 표시

### 3. 콘텐츠 품질 (Content Quality)

- ✅ **원본 콘텐츠**
  - 환율 계산기 (메인 기능)
  - 다양한 유틸리티 도구 (4개 이상)
  - FAQ 콘텐츠
  - 가이드 콘텐츠
  - 블로그 콘텐츠

- ✅ **저작권 표시**
  - Footer에 명확한 저작권 표시
  - "All rights reserved" 문구 포함

### 4. 기술적 요구사항 (Technical Requirements)

- ✅ **robots.txt**
  - 관리자 페이지 및 API 제외
  - Sitemap URL 포함

- ✅ **sitemap.xml**
  - 모든 필수 페이지 포함
  - FAQ, Guide 동적 콘텐츠 포함
  - 적절한 priority 및 changeFrequency 설정

- ✅ **메타데이터**
  - 모든 페이지에 적절한 title, description
  - Open Graph 태그
  - Twitter Card 태그
  - Canonical URL

### 5. Google AdSense 준비

- ✅ **AdSense 스크립트**
  - `components/growth-engine/GoogleAdSense.tsx` 구현
  - 조건부 로딩 (환경 변수 기반)

- ✅ **쿠키 정책**
  - Privacy Policy에 쿠키 사용 목적 명시
  - Google AdSense 쿠키 사용 안내
  - 쿠키 설정 거부 방법 안내

- ✅ **개인정보 처리방침**
  - Google AdSense 관련 내용 포함
  - 제3자 제공 안내

### 6. 사용자 경험 (User Experience)

- ✅ **모바일 최적화**
  - 반응형 디자인
  - 터치 친화적 인터페이스
  - 모바일 safe area 지원

- ✅ **접근성**
  - ARIA 레이블
  - 키보드 네비게이션 지원
  - 스크린 리더 지원

- ✅ **로딩 성능**
  - 이미지 최적화
  - 코드 스플리팅
  - 캐싱 전략

## 📋 승인 전 확인 사항

### 1. 콘텐츠 확인
- [ ] 모든 페이지에 충분한 원본 콘텐츠가 있는지 확인
- [ ] 중복 콘텐츠가 없는지 확인
- [ ] 저작권 침해 콘텐츠가 없는지 확인

### 2. 연락처 정보
- [ ] `site.config.ts`의 `author.email`이 실제 이메일 주소인지 확인
- [ ] Contact 페이지의 이메일이 실제로 작동하는지 확인

### 3. 도메인 및 배포
- [ ] 프로덕션 도메인으로 배포 완료
- [ ] HTTPS 적용 확인
- [ ] 모든 페이지가 정상적으로 로드되는지 확인

### 4. AdSense 설정
- [ ] Google AdSense 계정 생성
- [ ] `NEXT_PUBLIC_ADSENSE_CLIENT_ID` 환경 변수 설정
- [ ] AdSense 코드가 정상적으로 로드되는지 확인

### 5. 정책 준수
- [ ] 연령 제한 콘텐츠 없음 확인
- [ ] 폭력, 혐오, 선정성 콘텐츠 없음 확인
- [ ] 불법 콘텐츠 없음 확인
- [ ] 클릭 유도 문구 없음 확인

## 🚀 승인 신청 전 체크리스트

1. **콘텐츠 최소 요구사항**
   - ✅ 메인 페이지 (환율 계산기)
   - ✅ 최소 10개 이상의 고품질 페이지
   - ✅ FAQ 최소 5개 이상
   - ✅ 가이드 최소 3개 이상

2. **법적 페이지**
   - ✅ About 페이지 (상세)
   - ✅ Contact 페이지 (실제 연락처)
   - ✅ Privacy Policy (쿠키 정책 포함)
   - ✅ Terms of Service (상세)

3. **기술적 준비**
   - ✅ robots.txt
   - ✅ sitemap.xml
   - ✅ 모든 페이지 메타데이터
   - ✅ 모바일 최적화

4. **AdSense 준비**
   - ✅ AdSense 스크립트 통합
   - ✅ 쿠키 정책 명시
   - ✅ 개인정보 처리방침에 AdSense 내용 포함

## 📝 추가 권장 사항

### 1. 콘텐츠 확장
- 더 많은 FAQ 추가 (최소 10개 이상 권장)
- 더 많은 가이드 콘텐츠 추가
- 블로그 포스트 추가

### 2. 트래픽 확보
- SEO 최적화 (이미 완료)
- 소셜 미디어 공유
- 검색 엔진 제출

### 3. 사용자 참여
- 댓글 기능 (선택사항)
- 소셜 공유 버튼
- 피드백 수집

## ⚠️ 주의사항

1. **연락처 정보**
   - `site.config.ts`의 `author.email`을 실제 이메일 주소로 변경해야 합니다.
   - 현재는 `contact@example.com`으로 설정되어 있습니다.

2. **도메인 설정**
   - 프로덕션 배포 시 `NEXT_PUBLIC_SITE_URL` 환경 변수를 실제 도메인으로 설정해야 합니다.

3. **콘텐츠 품질**
   - 모든 콘텐츠가 원본이고 고품질이어야 합니다.
   - 복사-붙여넣기 콘텐츠는 승인을 거부당할 수 있습니다.

4. **정책 준수**
   - Google AdSense 정책을 정기적으로 확인하고 준수해야 합니다.
   - 정책 위반 시 계정이 정지될 수 있습니다.
